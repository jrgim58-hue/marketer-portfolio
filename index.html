문제
광고비는 늘었지만 매출 성장이 정체되고 ROAS가 하락. 비효율 타겟과 중복 노출로 예산이 낭비되고 있었습니다.
역할
퍼포먼스 캠페인 리드 (2024.04 ~ 2024.05)
실행
데이터 기반 퍼널 진단으로 문제 구간을 식별하고 채널·소재·타겟을 재설계. 구매 의도 세그먼트 확장, 크리에이티브 A/B 테스트, 입찰·예산 재배분.
성과
ROAS 320% 달성 (전 분기 대비 +220%p), 신규 고객 재구매율 동반 상승.
채널·툴
Google Ads, Meta Ads, 네이버 SA · GA4, Looker Studio, BigQuery
문제
리드당 획득 비용(CAC)이 업계 평균의 1.8배. 광고 소재와 랜딩 페이지 메시지가 어긋나 전환율이 낮았습니다.
역할
그로스 마케터 (2023.09 ~ 2024.01)
실행
퍼널 최적화와 소재-랜딩 메시지 매칭 테스트. 고의도 키워드 중심으로 예산 이동, 폼 단계 축소 실험.
성과
CAC 42% 절감, 동일 예산에서 SQL(영업 인계 리드) 1.6배.
채널·툴
LinkedIn Ads, Google Ads · GA4, HubSpot, SQL
문제
가입 직전 이탈이 60% 이상. 온보딩 메시지가 일괄 발송이라 사용자 상태와 맞지 않았습니다.
역할
CRM 마케터 (2023.02 ~ 2023.07)
실행
행동 데이터 기반 시나리오 분기 설계(방문·체험·결제 단계별), 발송 시점·카피 A/B 테스트 12회.
성과
신규 가입 전환 2.1배, 온보딩 완료율 +35%p.
채널·툴
Braze, 카카오 알림톡 · GA4, Amplitude
